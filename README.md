# MMHLSyn
This repository contains the code and data for "MMHLSyn:Synergistic prediction of drug combinations based on multi-view feature learning and multi-task supervision"
# Requirements
* python 3.7
* deepchem >= 2.5
* numpy >= 1.19
* pandas >= 1.3
* pytorch >= 1.8.0
* pytorch geometric >= 2.0.0 
* scikit-learn >= 1.0.2
* rdkit >= 2020.09.1
